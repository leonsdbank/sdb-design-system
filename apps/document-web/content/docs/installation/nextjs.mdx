## Prerequisites

- Next.js 16+
- Tailwind CSS 4
- React 19+

## Install Dependencies

```bash
bun add @sdbank/ui react-native-web@^0.21.0
```

```bash
bun add -d tailwindcss @tailwindcss/postcss
```

## Configure Next.js

Wrap your Next.js config with `withSdbUI()`. This configures Turbopack aliases, CSS layer ordering, and `transpilePackages` automatically.

```ts title="next.config.ts"
import { withSdbUI } from "@sdbank/ui/next";

export default withSdbUI({
  // your Next.js config here
});
```

You can pass options as a second argument:

```ts title="next.config.ts"
import { withSdbUI } from "@sdbank/ui/next";

export default withSdbUI(
  {
    // your Next.js config
  },
  {
    extraTranspilePackages: ["other-rn-package"],
  }
);
```

## Configure PostCSS

```js title="postcss.config.mjs"
const config = {
  plugins: {
    "@tailwindcss/postcss": {},
  },
};

export default config;
```

## Configure CSS

Import the `@sdbank/ui` global CSS **before** Tailwind. This sets up the `@layer rnw` ordering so Tailwind utilities correctly override react-native-web base styles.

```css title="globals.css"
@import "@sdbank/ui/global.css";
@import "tailwindcss";

@source "../../node_modules/@sdbank/ui/dist/src";
```

## Configure TypeScript

Add the `@sdbank/ui` type augmentation to your `tsconfig.json` so the `className` prop is recognized on React Native components:

```json title="tsconfig.json"
{
  "compilerOptions": {
    "paths": {
      "~/*": ["./src/*"]
    }
  },
  "include": [
    "next-env.d.ts",
    "**/*.ts",
    "**/*.tsx",
    ".next/types/**/*.ts",
    "node_modules/@sdbank/ui/dist/src/types/**/*.d.ts"
  ]
}
```

## Usage

```tsx title="app/page.tsx"
import { Button, Text, Card, CardHeader, CardTitle, CardContent } from "@sdbank/ui";

export default function Page() {
  return (
    <Card className="m-4">
      <CardHeader>
        <CardTitle>Hello</CardTitle>
      </CardHeader>
      <CardContent>
        <Text className="text-gray-600 mb-4">
          @sdbank/ui components in Next.js
        </Text>
        <Button>Click me</Button>
      </CardContent>
    </Card>
  );
}
```

## How It Works

`withSdbUI()` handles all the wiring under the hood:

1. **Turbopack alias** — Maps `react-native` imports to a shim that bridges `className` to react-native-web's `style` prop using the `$$css` pattern.
2. **CSS layer ordering** — A loader patches react-native-web's stylesheet to use `@layer rnw`, preventing base styles from overriding Tailwind utilities.
3. **Webpack fallback** — Prepends `.web.*` extensions and aliases `react-native` for `next dev --webpack` compatibility.
4. **transpilePackages** — Adds `@sdbank/ui` and `react-native-web` so Next.js processes them correctly.
