## Introduction

**@sdb/ui** is a React Native component library built with [Uniwind](https://github.com/nicepkg/uniwind) (Tailwind CSS for React Native). It provides a set of composable, accessible components that work across React Native and Web.

## Features

- React Native components with Tailwind CSS styling via Uniwind
- Cross-platform support (iOS, Android, Web)
- TypeScript-first with full type safety
- `className` prop support on all components
- Composable component patterns (e.g., Card system)

## Installation

Choose your framework to get started:

- [Next.js](/docs/installation/nextjs) — Full setup with `withSdbUI()` config wrapper
