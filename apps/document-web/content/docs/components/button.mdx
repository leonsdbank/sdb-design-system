---
title: Button
description: An interactive button component with variant and size support.
---

## Preview

<ButtonPreview />

## Usage

```tsx
import { Button } from '@sdbank/ui';

export default function Example() {
  return <Button>Click me</Button>;
}
```

## Variants

```tsx
<Button variant="default">Default</Button>
<Button variant="secondary">Secondary</Button>
<Button variant="destructive">Destructive</Button>
<Button variant="outline">Outline</Button>
<Button variant="ghost">Ghost</Button>
```

## Sizes

```tsx
<Button size="sm">Small</Button>
<Button size="default">Default</Button>
<Button size="lg">Large</Button>
<Button size="icon">Icon</Button>
```

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `variant` | `'default' \| 'secondary' \| 'destructive' \| 'outline' \| 'ghost'` | `'default'` | Visual style variant |
| `size` | `'default' \| 'sm' \| 'lg' \| 'icon'` | `'default'` | Size of the button |
| `className` | `string` | - | Additional Tailwind classes |
| `textClassName` | `string` | - | Tailwind classes for the inner text |
| `disabled` | `boolean` | `false` | Disables the button |
